# 项目架构

```bat
main
 ├────java----主代码目录
 		├────com.may----项目目录
                ├────constant----常量配置层
 				├────controller----控制层
 				├────dao----数据访问层(mapper):数据库持久化操作
 				├────dto----数据传输层
 				├────exception----错误信息层
 				├────interceptor----拦截器
 				├────log----日志文件
 				├────model----数据模型层(entity、domain):存放映射到数据库的实体类
 				├────service----服务层
 						└────impl----接口层
                └────com.may-utils----工具层
 └────resources----资源目录
 		├────i18n----国际化
 		├────static----静态文件
        		├────css
                ├────images
                ├────js
                └────lib
 		├────templates----网页模板
        		├────admin----后台管理
                ├────blog----前端页面
                ├────commons----通用模板
                └────error----错误页面
 		├────application.yaml----环境配置文件
 		├────application-dev.yaml----开发环境
 		├────application-prod.yaml----生产环境
 		├────application-test.yaml----测试环境
 		└────banner.txt
```
